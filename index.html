Instrucciones

Setup por herramienta

1

Entrá a Gemini con tu cuenta de Nomadic

Abrí gemini.google.com e iniciá sesión con tu email de Nomadic. El acceso ya está incluido en el Workspace de la empresa.

2

Subí los 6 archivos de contexto a Google Drive

Creá una carpeta llamada Nomadic IA Setup en tu Drive y subí los 6 archivos .md que descargaste. Los Gems referencian archivos desde Drive y siempre usan la versión más actualizada.

3

Creá un Gem llamado "Nomadic Assistant"

En Gemini, hacé clic en Explorar Gems → Gem nuevo. Dale el nombre Nomadic Assistant. Los Gems son versiones configuradas de Gemini que mantienen el contexto en todas tus conversaciones — el equivalente a los Projects de Claude.

4

Agregá las instrucciones base

En el campo de instrucciones del Gem, pegá lo siguiente:

Sos un asistente de trabajo para el equipo de Nomadic. Usá siempre el contexto de los archivos adjuntos para entender quiénes somos, cómo trabajamos y cómo comunicamos. Respondé en español rioplatense, con voseo. Sé directo y sin relleno.
5

Adjuntá los archivos desde Drive

En la sección de archivos del Gem, hacé clic en Drive y seleccioná los 6 archivos de contexto que subiste. Gemini los va a leer automáticamente en cada conversación.

6

Guardá y probá

Hacé clic en Guardar. Ahora usá siempre este Gem en lugar del chat general de Gemini. Podés acceder desde el sidebar izquierdo en Mis Gems.

✦ Por dónde arrancar
Preguntale al Gem cómo usarlo
Una vez que configuraste el Gem, no necesitás saber nada más para empezar. Pegá uno de estos prompts y dejá que te guíe.
¿Qué podés hacer por mí como integrante del equipo Nomadic? Listame 5 casos de uso concretos para mi rol.
Leé los archivos de contexto que tenés cargados y explicame en tres oraciones qué sabés sobre Nomadic.
Ayudame a escribir un resumen ejecutivo de una reunión. Primero preguntame qué información necesitás.
1

Descargá Antigravity

Antigravity (antigravity.dev) es una herramienta de agentes de IA del ecosistema Google, disponible en el tier incluido con tu cuenta. Ideal para tareas que van más allá del chat — automatizaciones, procesamiento de archivos y flujos de trabajo.

2

Creá una carpeta de trabajo y copiá los archivos

Creá una carpeta llamada nomadic-workspace en tu computadora:

Mac / Linux → /Users/tu-usuario/nomadic-workspace/ Windows → C:\Users\tu-usuario\nomadic-workspace\

Copiá los 6 archivos descargados adentro. Cuando abrís Antigravity apuntás a esa carpeta — y el archivo GEMINI.md se carga automáticamente como instrucciones globales.

3

Copiá y pegá al arrancar
Leé los archivos GEMINI.md, about-nomadic.md, working-preferences.md, brand-guidelines.md y safety.md de esta carpeta. 1

Instalá Node.js LTS

Descargá desde nodejs.org → versión LTS. Verificá que quedó bien:

node --version # Tiene que aparecer v20.x.x o superior

¿Dónde está la terminal? Mac: Spotlight (Cmd+Espacio) → "Terminal". Windows: Menú inicio → "PowerShell".

2

Instalá Claude Code

npm install -g @anthropic-ai/claude-code
3

Autenticate

claude

La primera vez abre el navegador para loguearte con tu cuenta Claude Pro. Hacés login y volvés a la terminal.

4

Descargá Cursor o VSCode (gratuitos)

Cursor — cursor.com  ·  VSCode — code.visualstudio.com

Extensiones (Cmd+Shift+X / Ctrl+Shift+X) → buscá Claude Code → instalá la extensión oficial de Anthropic.

5

Abrí el workspace en el IDE

File → Open Folder → ~/nomadic-workspace/. El CLAUDE.md tiene que estar en la raíz para que Claude Code lo lea automáticamente.

6

Lanzá Claude Code desde la terminal del IDE

cd ~/nomadic-workspace && claude
7

Modos de operación

Normal — pide confirmación antes de cada acción.
Auto-Accept (Shift+Tab × 1) — ejecuta sin confirmación.
Plan Mode (Shift+Tab × 2) — planifica sin ejecutar.

Flujo recomendado: Plan Mode → revisás el plan → Auto-Accept.

8

Comandos útiles en sesión

/compact → compactar contexto en sesiones largas /clear → limpiar conversación y arrancar de cero /exit → cerrar la sesión

Referencia

Glosario rápido

Agente
Una IA que no solo responde preguntas sino que toma acciones: lee archivos, ejecuta código, usa herramientas externas. Antigravity y Claude Code son agentes. Gemini Chat y Claude Chat no lo son.
MCP Model Context Protocol
Estándar abierto (creado por Anthropic, adoptado por Google) que permite conectar una IA con servicios externos — Slack, Drive, bases de datos, APIs. Es como un puerto USB-C para herramientas de IA: una vez que algo tiene MCP, funciona con cualquier agente compatible. Disponible en Antigravity y Claude Code.
Skill
Un paquete de instrucciones especializadas para el agente — una carpeta con un archivo SKILL.md que le enseña cómo encarar un tipo de tarea concreta. Por ejemplo: un skill para redactar briefs, otro para analizar métricas, otro para hacer code reviews. Disponible en Antigravity y Claude Code.
Contexto
Todo lo que la IA "sabe" en una sesión: los archivos cargados, las instrucciones configuradas, el historial de conversación. Los archivos de este setup son contexto — le dan a la herramienta información sobre Nomadic antes de tu primera pregunta.
Prompt
El mensaje o instrucción que le das a la IA. Un buen prompt es específico, da contexto y dice qué formato de respuesta esperás. La diferencia entre un prompt vago y uno bien construido puede cambiar completamente la calidad del output.
Plan Mode / Planning Mode
Modo de los agentes (Claude Code y Antigravity) en el que la IA planifica lo que va a hacer pero no ejecuta nada hasta que vos lo aprobás. Esencial para tareas que modifican archivos o conectan con servicios externos.
Git
Sistema de control de versiones que guarda el historial completo de cambios de cualquier proyecto. Si algo se rompe, podés volver a cualquier punto anterior. Estándar para trabajo colaborativo en código.
GitHub
La plataforma donde viven los repositorios Git — como un Google Drive pero para proyectos versionados. Es donde Nomadic centraliza el código, los setups de IA y la documentación técnica compartida.
